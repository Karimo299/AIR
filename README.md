# Airpods ++

Add more actions to your AirPods!

Set multiple actions for a double tap and quadruple tap.

***Supports iOS 11***
## Actions

* Pause/Play.

* Skip.

* Rewind.

* Skip15.

* Rewind15.

* IncreaseVolume.

* DecreaseVolume.

* ToggleSiri.
